Gérer

Optimiser une base de données, interagir avec une application et mettre en œuvre la sécurité

Projets :

  • Dans le cours R3.07 SQL ddans un langage de programmation, nous avons vu comment utiliser des index pour optimiser les recherches dans une base de données relationnel, nous avons aussi vu comment assurer l'intégrité des données en utilisant des transactions
  • Dans le projet facture nous avons pu mettre en place une migration de base de données en utilisant des procédures stockée et en ajoutant des trigger pour s'assurer de l'intégrité des données
  • Avec le cours R4.03 Qualité au-delà du relationnel nous avons découverts les base de données non relationnel avec MongoDB, nous avons pu mettre en pratique grace au projet TranWay qui utilise une base de donnée non-relationnel

Administrer

Déployer des services dans une architecture réseau

  • Dans la matière R3.06 nous avons mis en place et configurer des différents serveur (web, proxy...) sur des machines virtuelle en TP.
  • Dans la matière R3.05 Programmation système, nous avons pu découvir comment les application peuvent communiquer entre elles en utilisant des mécanismes bas niveaux (pipes, fichier, stdout...) nous avons pratiqué avec des TP.
  • Dans le projet TranWay nous avons utilisé des token JWT pour sécuriser la communication entre le front-end et l'API.
  • Pendant le cours R4.08 Virtualisation, nous avons étudié les algorithmes de chiffrement (symétrique/asymétrique, Diffie-Hellman) utilisé pour sécuriser les communication.
